## 学到的点

1. **Astro + Cloudflare 适配器的限制**: 当 `output: 'static'` 时，无法在服务端读取本地文件（fs/path 模块被禁用）
2. **Drizzle ORM + Neon**: 需要 `DATABASE_URL` 环境变量连接到 Neon 数据库
3. **dotenv-cli**: 有 Python 和 Node.js 两个版本，需要使用 `npx dotenv` (Node.js 版本)
4. **Mock 数据库**: 在开发环境下使用 mock 可以无需数据库凭证进行前端测试
