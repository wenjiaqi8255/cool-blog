  <action>
Create src/lib/mcp/db.ts with 4 functions:

1. createArticle(data: CreateArticleInput): Promise<Article>
   - Input: { title, body, date?, tags?, excerpt?, status? }
   - Generate slug via generateSlug(title) from slugify.ts
   - Handle slug collision: append -1, -2, etc. if duplicate
   - Insert into articles table using Drizzle
   - Return created article

2. listArticles(options: ListArticlesOptions): Promise<Article[]>
   - Input: { status?, limit?, offset?, order_by? }
   - Query articles where deleted_at IS NULL (exclude soft-deleted)
   - Filter by status if provided
   - Order by date DESC or ASC
   - Apply limit (default 20) and offset (default 0)
   - Return array of articles

3. getArticle(slug: string): Promise<Article | null>
   - Input: slug (required)
   - Query articles where slug = input AND deleted_at IS NULL
   - Return article or null if not found

4. deleteArticle(slug: string): Promise<boolean>
   - Input: slug (required)
   - Update articles set deleted_at = NOW() where slug = input AND deleted_at IS NULL
   - Return true if updated, false if not found

Use exact Drizzle patterns from RESEARCH.md:
- Import: eq, desc, asc, isNull, and from drizzle-orm
- Query: db.query.articles.findMany({ where: ..., orderBy: ..., limit, offset })
- Insert: db.insert(articles).values(...).returning()
- Update: db.update(articles).set({ deleted_at: new Date() }).where(eq(articles.slug, slug))
  </action>
