## Component Inventory

For gsd-planner and gsd-executor:

### Article Card (Bento)
- Bento card variant matching Portfolio visual language
- Shows: cover image (if available), title, excerpt, date, tags
- Hover: same animation pattern as Portfolio cards
- Grid: 4-col → 2-col → 1-col responsive (same as Phase 1)

### Article Page
- Content width: 800-900px centered
- Article header: title, reading time, date, tags (no cover image)
- Body: Inter 16px, 1.5 line-height
- Code blocks: #111 background, JetBrains Mono 14px, copy button

### Tag Filter
- Pill buttons at top of article list
- OR logic: any selected tag matches
- States: default (outline), selected (filled #111)

### Search Input
- Client-side search (Fuse.js)
- Searches title and content
- Shows results via infinite scroll

### Infinite Scroll
- Load more as user scrolls
- No pagination buttons
